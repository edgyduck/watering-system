# watering-system
Plants watering system based on Arduino

Purpose: create poor plants watering automation and save them from death.

Used Material

Required:
- Arduino Nano v3
- RTC module
- Water pump (5V)
- Water level sensor
- Moisture sensor
- 12V battery
- Water hose (4 mm)
- Plants
- Wires

Optional:
- Wire jumpers
- Plastic box
- Ice cream sticks
- Hot glue
